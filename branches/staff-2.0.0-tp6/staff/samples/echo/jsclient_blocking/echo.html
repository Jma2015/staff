To start this client:

  1. start codegen to produce client proxy from current directory:
    staff_codegen -tjsclient -i../client_blocking/src Echo.h -dserviceuri=http://localhost/axis2/services/echo
  2. download jQuery from http://jquery.com/ site and put it as jquery.js into this directory
  3. copy staff.js from directory staff/bindings/javascript/client into this directory
  4. deploy current directory on web server
  5. click the button
  6. you should see message box with response from service
